Join Hackital's 2020 Game Jam. Our Game Jam is an online, 2 week long event, where we challenge everyone to build and launch a game from scratch using any technology, around a theme that we will announce the day the event kicks off. We will host several free online technical workshops on programming and game building over the course of the jam for complete beginners and veterans alike. The Game Jam is open to everyone in the world. After you sign up we will invite you to a private Discord to chat with fellow participants and gain access to the workshops.
